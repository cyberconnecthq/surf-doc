## Overview

Returns research content for a given ticker including title and detailed content. Both ticker (path parameter) and template_id (query parameter) are required.

## API Details

**URL:** `GET https://api.asksurf.ai/huginn/v2/chat/research/{ticker}`

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| ticker | string (path) | Yes | Ticker symbol |
| template_id | string (query) | Yes | Template ID |

### Response

Returns a `GenericResponse[ResearchResponse]` object with the following structure:

```json
{
  "success": true,
  "message": "success",
  "error_code": "",
  "data": {
    "title": "string",
    "content": "string"
  }
}
```

### Response Status Codes

- **200**: Returns research information on success
- **400**: When ticker or template_id is missing
- **500**: Internal server error

<RequestExample>

```bash cURL
curl -X GET "https://api.asksurf.ai/huginn/v2/chat/research/BTC?template_id=template-123" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const ticker = "BTC";
const templateId = "template-123";

const response = await fetch(
  `https://api.asksurf.ai/huginn/v2/chat/research/${ticker}?template_id=${templateId}`,
  {
    method: 'GET',
    headers: {
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

ticker = "BTC"
template_id = "template-123"

url = f"https://api.asksurf.ai/huginn/v2/chat/research/{ticker}"
params = {"template_id": template_id}

response = requests.get(url, params=params)
data = response.json()
print(data)
```

</RequestExample>

## Notes

- Both ticker and template_id parameters are required
- The research content is dynamically generated based on the template and ticker
- Content may include market analysis, fundamental analysis, technical indicators, and other relevant information
