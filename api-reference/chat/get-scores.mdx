## Overview

Returns various scores for a given ticker including average score, smart follower count score, mind share score, and cyber score. Both ticker (path parameter) and template_id (query parameter) are required.

## API Details

**URL:** `GET https://api.asksurf.ai/huginn/v2/chat/scores/{ticker}`

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| ticker | string (path) | Yes | Ticker symbol |
| template_id | string (query) | Yes | Template ID |

### Response

Returns a `GenericResponse[GetChatScoreResponse]` object with the following structure:

```json
{
  "success": true,
  "message": "success",
  "error_code": "",
  "data": {
    "score": 0.0,
    "smart_followers_count_score": 0.0,
    "mind_share_score": 0.0,
    "cyber_score_score": 0.0
  }
}
```

### Response Status Codes

- **200**: Returns chat score information on success
- **400**: When ticker or template_id is missing
- **500**: Internal server error

<RequestExample>

```bash cURL
curl -X GET "https://api.asksurf.ai/huginn/v2/chat/scores/BTC?template_id=template-123" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const ticker = "BTC";
const templateId = "template-123";

const response = await fetch(
  `https://api.asksurf.ai/huginn/v2/chat/scores/${ticker}?template_id=${templateId}`,
  {
    method: 'GET',
    headers: {
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

ticker = "BTC"
template_id = "template-123"

url = f"https://api.asksurf.ai/huginn/v2/chat/scores/{ticker}"
params = {"template_id": template_id}

response = requests.get(url, params=params)
data = response.json()
print(data)
```

## Notes

- Both ticker and template_id parameters are required
- The score values range from 0.0 to 1.0
- All score metrics are calculated based on various data sources and algorithms
